Instructions

 This is a prototype application for analyzing sentiment of documents from a corpus of opinionated text. The tool will guide you through the following steps:

  1. Corpus selection

    In this step, you can pick one of the following options:

    1. Upload a corpus

      You can upload corpora in CSV, text, or XML formats. In the case of CSV and text files, each line should contain a separate review or comment. XML files should have a root element with any number of child nodes where each node would be considered a separate review or comment. You can also upload a ZIP file containing several files in one of the above formats.

    2. Load an already uploaded corpus

      You can access one of the corpora that you have uploaded recently in this session. There is no guarantee that a corpus you uploaded earlier will be available to you if you close your browser window.

    3. Retrieve a corpus with a key

      This is an advanced option. If you have to close your browser window and you want a better chance of coming back to the same page, you can enter the UUID-style key from one of the later steps. Again, there is no guarantee this will work, but it's an option to try.

  2. Reduce the size of corpus to annotate

    This is only applicable when uploading a new corpus or reusing an unfiltered corpus. You will be able to select a threshold for data reduction. More reduction means less data for you to look at, and a higher threshold means higher loss in data quality.

  3. Aspect extraction

    In this step, you can create an ontology for your domain by adding terms to the aspects and keywords lists. Some words have been highlighted which the tool thinks are more likely to be keywords; these can be clicked or dragged and dropped into one of the lists as convenient. Please note that the number and quality of keywords and aspects selected here will affect the performance of the opinion mining module. (The URL for this page contains a UUID-style key that can be used in the first step to retrieve this corpus.)

  4. Opinion analysis

    Finally, you can browse through individual comments and the breakdown of sentiments for various aspects.
Please remember that this is a prototype application so we would really appreciate your feedback should you experience any problems.

Requirements
